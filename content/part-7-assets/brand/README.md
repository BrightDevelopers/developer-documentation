# BrightSign Brand Assets

This directory contains official BrightSign brand assets for use within this documentation.

## Logo Files

| File | Usage |
|------|-------|
| `brightsign-logo.png` | Primary logo for headers and documentation |
| `brightsign-logo-square.png` | Square format for footers and icons |

## Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| BrightSign Purple | `#311C6B` | Primary brand color |
| White | `#FFFFFF` | Backgrounds, text on dark |
| Ultraviolet | `#701896` | Secondary accent |
| Highlighter Blue | `#49C0FF` | Links, highlights |
| Soft Black | `#1C1C1C` | Text, dark backgrounds |

## Typography

- **Primary Font**: Montserrat
- **Fallback**: Sans-serif

## Usage Guidelines

- Maintain logo aspect ratio
- Ensure sufficient contrast with backgrounds
- Use purple logo on light backgrounds
- Use white logo on dark backgrounds

For complete brand guidelines, visit [brightsign.biz](https://brightsign.biz).
